Synopsis

create a namespace

circleci namespace create NAME VCS-TYPE ORG-NAME [flags]

Arguments

NAME        The name to give your new namespace
VCS-TYPE    Your VCS provider, can be either "github" or "bitbucket"
ORG-NAME    The name used for your organization

Flags

  -h, --help   help for create

Flags inherited from parent commands

      --host string    URL to your CircleCI host (default "https://circleci.com")
      --token string   your token for using CircleCI
      --verbose        Enable verbose logging.
